## 🎨 2. סגנון העיצוב — "Dark Tech"

### פלטה

```css
--bg: #080a0f;          /* רקע ראשי */
--bg2: #0d1018; --bg3: #121620; --bg4: #161b28;
--blue: #3a9fff;        /* כחול מותג */
--blue2: #1a6fd8; --blue-glow: rgba(58,159,255,0.12);
--accent: #00d4ff;      /* תכלת זוהר */
--gold: #F5A623;        /* זהב (חבילות, CTA) */
--wa: #25D366;          /* ירוק וואטסאפ */
--text: #e8eaf0; --text2: #9aa0b4; --text3: #5a6070;
```

### טיפוגרפיה
- **Display:** Rajdhani (400-700) — כותרות, מספרים, לוגו
- **Body:** Noto Sans Hebrew (300-500) — טקסט
- כיוון: RTL (אך הלוגו עם `dir="ltr"` כדי שיקרא "SafeView")

### עקרונות
- רקע כהה
- אנימציות עדינות (pulse, fadeUp, scroll reveal)
- borders דקים עם opacity
- "//" כסימן לפני תוויות sections (סגנון terminal)
- אנגלית טכנית משולבת (LIVE, 2K, IP67)
- **אייקוני SVG אחידים** — לא אימוג'ים (מלבד דגלי לאום)

## 🏗️ 4. מבנה האתר (HTML)

```
/
├── index.html              ← דף בית: hero, קטגוריות, 9 מוצרים, 3 חבילות, יתרונות, FAQ
├── product.html?id=<id>    ← עמוד מוצר דינמי: גלריה, מפרט, ביקורות, מוצרים קשורים
├── compare.html            ← דף השוואה מסונן לפי קטגוריה
├── cart.html               ← עגלה עם localStorage + צ'קאאוט בוואטסאפ
├── about.html              ← אודות
├── contact.html            ← צור קשר
├── faq.html                ← שאלות נפוצות עם accordion + FAQ Schema
├── blog.html               ← מאמרי מדריך
├── shipping.html           ← מדיניות משלוחים
├── returns.html            ← החזרות והחלפות
├── privacy.html            ← פרטיות
├── terms.html              ← תקנון
├── 404.html                ← דף שגיאה מותאם
├── assets/
│   ├── style.css           ← כל ה-CSS (Dark Tech + responsive + a11y)
│   └── app.js              ← קטלוג, עגלה, וואטסאפ, nav/footer injection, FAQ, a11y, cookies
├── manifest.json           ← PWA installable
├── sw.js                   ← Service Worker (cache strategy)
├── favicon.svg             ← לוגו זמני
├── sitemap.xml             ← SEO
├── robots.txt              ← SEO
├── .gitignore              ← קבצים שלא נכנסים ל-repo
├── CLAUDE.md               ← זיכרון לסשנים עתידיים (קצר)
└── SafeView_CONTEXT.md     ← מסמך הקשר מורחב (זה הקובץ)
```

### פיצ'רים חיים באתר:
- 🛒 **עגלת קניות** (localStorage) עם בר התקדמות למשלוח חינם
- 🟢 **כפתורי וואטסאפ** — לכל מוצר, צף, וצ'קאאוט מלא של ההזמנה
- 📱 **רספונסיב מלא** — תפריט המבורגר במובייל, sticky add-to-cart במובייל בדף מוצר
- 🎯 **אנימציות גלילה** (IntersectionObserver)
- 🔍 **SEO** — sitemap, robots, JSON-LD (Organization, Store, Product, Breadcrumbs, FAQ), canonical/OG אוטומטי
- 🎨 **אייקוני SVG אחידים** ב-`ICON` במקום אימוג'ים
- ♿ **פאנל נגישות צף** — גודל פונט, ניגודיות, הדגשת קישורים, צמצום אנימציות
- 🍪 **באנר עוגיות** עם הסכמה (localStorage)
- 📦 **PWA** — האתר ניתן להתקנה כאפליקציה
- 🔒 **תגי אמון** ב-footer + אמצעי תשלום

## 🎓 11. כללי עבודה — חשוב לזכור

1. **שמור על סגנון Dark Tech** — אותה פלטה, אותם פונטים תמיד
2. **תמיד עברית + RTL**, אנגלית טכנית מינימלית (LIVE, 2K, IP67)
