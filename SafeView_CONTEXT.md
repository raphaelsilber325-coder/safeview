## 3. מערכת העיצוב – "Dark Tech"

### פלטה
```css
--bg: #080a0f;          /* רקע ראשי */
--bg2: #0d1018;         /* רקע משני */
--bg3: #121620;         /* כרטיסים */
--bg4: #161b28;         /* hover */
--blue: #3a9fff;        /* צבע מותג ראשי */
--blue2: #1a6fd8;       /* כחול כהה */
--blue-dim: #1a3560;    /* כחול דהוי */
--blue-glow: rgba(58,159,255,0.12);
--accent: #00d4ff;      /* תכלת זוהר */
--text: #e8eaf0;        /* טקסט ראשי */
--text2: #9aa0b4;       /* טקסט משני */
--text3: #5a6070;       /* טקסט שלישוני */
--border: rgba(58,159,255,0.15);
--border2: rgba(255,255,255,0.06);
```

### טיפוגרפיה
- **Display:** `Rajdhani` (400-700) – לכותרות, מספרים ולוגו
- **Body:** `Noto Sans Hebrew` (300-500) – לטקסט כללי
- כיוון: RTL

### עקרונות עיצוב
- רקע כהה עם רשת עדינה (`grid overlay` של נקודות)
- אנימציות sleek: scan line, pulse, fadeUp
- borders דקים עם opacity (0.5px - 1px)
- אקצנטים כחולים זוהרים ב-hover
- אייקונים חד-צבעוניים
- אנגלית טכנית (LIVE, 2K, IP67) משולבת בעברית
- "//" כסימן לפני תוויות sections (סגנון terminal)

### רכיבים שמומלץ לבנות
- Hero עם תצוגת מצלמה אנימטית (scan line + LIVE badge)
- Sticky navbar עם blur
- כרטיסי מוצר עם תמונה אמיתית, badge קטגוריה ומחיר ב-Rajdhani
- Features ממוספרים (01, 02, 03) בפונט Rajdhani
- CTA band עם רדיאל gradient עדין

## 7. הוראות לקלוד קוד

כאשר ממשיכים בפרויקט הזה:

1. **שמור על סגנון Dark Tech** בכל הקבצים החדשים (אותה פלטה, אותם פונטים)
2. **תמיד RTL ובעברית** עם דגש על אנגלית טכנית מינימלית
3. **תמונות אמיתיות** – יש בקובץ הזה קישורים ל-CDN של שופיפיי
4. **Performance first** – אל תוסיף libraries כבדים, השאר vanilla JS
5. **Mobile-first** – הרבה מהלקוחות יגיעו מהטלפון
6. **הקפד על תאימות עם Shopify Liquid** אם בונים עבור החנות
